frontend-nanodegree-arcade-game
===============================
## Keyboard Control
Use Up, Right, Down, Left to control the player move.

## Player select
Move the player to the character to select.

## Score
1. Reach the river get 100.
2. Crash a bug minus 10.
3. Collect gems get extra score:
	- Blue: 500
	- Green: 2000
	- Orange: 5000
